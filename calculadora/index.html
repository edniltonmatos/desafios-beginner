Calculadora app - html, css e JavaScript

A calculadora app é uma calculadora comum criada com html, css e java script. Ela te permite fazer operações básicas de soma, subtração, divisão e adição.

Esse projeto foi baseado na ideia do repositorio App Ideas Collection.
